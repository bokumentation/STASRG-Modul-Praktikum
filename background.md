# Background

Saya ketua panitia praktikum sekaligus seleksi untuk mahasiswa riset baru lab kampus.
Saya ditugaskan untuk menyusun modul berupa pengenalan solder dan esp32 menggunakan arduino ide.

Modul praktikum hardware akan lab STAS-RG akan saya pisah menjadi:
- Pengenalan Solder di papan PCB.
    - Peserta diminta untuk merakit ESP32, Push Button dengan pull up, Buzzer 3.3v , 2 LED with 330ohm resistors, sensor MQ-7, DHT22, Pizoelektrik, ADXL345. Menggunakan PCB matrix hole dan kabel dupont 2.54mm female. Masing-masing wajib menggunakan pinheader male dan female, sehingga sambungan menggunakan kabel jumper tanpa solder ke sensor.  
- Pengenalan ESP32S3 menggunakan Arduino IDE.
    - Peserta diminta membuat kode dan memflash ke esp32s3 berupa: kode untuk LED Blinking, Push button untuk menyalakan led dan mengatur led running, membaca sensor AHT21 dan mengeluarkannya ke serial monitor dengan format tertentu.

Peserta diminta untuk berkelompok yang terdiri dari 2 orang. Maka total kelompok ada 14.
