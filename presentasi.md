   - Safety procedures

2. **Modul 2**: Pemrograman ESP32S3
   - Arduino IDE setup
   - GPIO control (LED, button, buzzer)
   - Sensor AHT21 (temperature & humidity)

3. **Modul 3**: GitHub Desktop
   - Version control basics
   - Repository management
   - Documentation dengan README.md

## Slide 5: Modul 1 - Soldering (Detail)
### Komponen yang Dirakit:
- ESP32S3 development board
- 3x LED dengan resistor 330Ω
- Push button dengan pull-up
- Buzzer 3.3V
- Sensor AHT21 (via pinheader)
- PCB matrix hole

### Teknik yang Dipelajari:
- Proper soldering technique
- Component placement
- Safety protocols
- Quality inspection

## Slide 6: Modul 2 - ESP32S3 Programming (Detail)
### Program yang Dibuat:
1. **LED Blinking** - Dasar GPIO output
2. **Button Control** - Digital input dengan debouncing
3. **Running LED** - Pattern control dengan multiple LEDs
4. **AHT21 Sensor** - I2C communication & data reading
5. **Complete Project** - Integrasi semua fitur

### Tools:
- Arduino IDE 2.0+
- ESP32 Board Package
- Adafruit AHT21 Library

## Slide 7: Modul 3 - GitHub Desktop (Detail)

4. **Documentation**: Create comprehensive README.md

- Professional project presentation
